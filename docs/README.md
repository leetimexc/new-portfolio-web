## 开发文档

该项目记录我从零开始开发个人简历网站的全过程，包括技术选型、开发流程、前后端开发、部署等。

---

## 技术栈

- 前端：astrojs
- 管理后台：element-plus
- 后端
  - jdk: java21
  - framework: springboot
  - build tool: maven
  - database: mysql
  - cache: redis
  - server: nginx
  - 下列工具组成自动化部署前缀链路：
    - ftp
    - aliyun
    - oss
    - github
    - git
    - jenkins
    - docker
